Join app authors and many thousands of users

Welcome!

We're so glad you're interested in getting involved! We're a movement of app makers, users, open source enthusiasts, self-hosting fanatics, enterprise IT staffers, documentation writers, and more, working together on making server apps safe and easy for everyone to use. Here's how you can contribute.

Help others

  • Use & Share Sandstorm

    Install an app from the App Market and leave a review. Create a grain, and invite a friend to collaborate.

  • Give a Talk

    Submit a talk to an open source meetup near you! Use our speaker kit. We can help you find a venue, help with talk prep, and send you sandcat stickers that you can give out! Send us an email!

  • Discuss & Answer Questions

    Development and user help happens on the sandstorm-dev group; join us, listen, and ask questions! Real-time chat happens in IRC (#sandstorm on freenode), with chat archives available too.

  • Build Local Community

    If you're in San Francisco, meet core developers, app authors, and Sandstorm enthusiasts at the Sandstorm SF Bay Area meetup group. If not, create a meetup group near you. The first meeting can be a small group over coffee. Email us to get started.

  • Write a Blog Post

    The world needs people talking about their own self-hosting stories. Write a blog post about something you've done with Sandstorm. We'll Tweet a link and get you some readers! Email us for topic ideas.

Help the project

  • Package an App

    Package an open source app that you enjoy running on your own server. The packaging tutorial will get you started. If you get stuck, email us for help or hop on IRC.

  • Provide Feedback

    For the apps you use, think about how they could be better. You can report issues against apps by visiting the Sandstorm app market, clicking on the app, and clicking on the Code link. You can report bugs in Sandstorm itself on our issue tracker.

  • Design App Icons

    Many apps that are being packaged for Sandstorm are in need of logos or icons so they can be displayed in our App Market. If you have design skills and would love to help out an app with graphics, send us an e-mail!

  • Usability Test for Sandstorm

    A usability test is where you ask a friend to try to do something with Sandstorm while you record their screen & face. We'd love your help doing a user test. Read Usability 101 by Jakob Nielsen then email us!

  • Read & Improve the Documentation

    The Sandstorm documentation lives in the main GitHub repository for Sandstorm. Listen to IRC and the dev group, identify a common question, then submit a pull request to the docs!

  • Contribute to Core

    The Sandstorm open source code lives in GitHub. To contribute bug reports, patches, design ideas, documentation, and pull requests check out the GitHub repository!
